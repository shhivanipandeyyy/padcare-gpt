PadCare Labs is a sustainability-focused startup working on menstrual waste management.

Problem:
India generates a large amount of sanitary waste every day. Most used sanitary pads end up in landfills or water bodies, which harms the environment and poses health risks to people handling waste.

Solution:
PadCare provides a closed-loop system for managing used sanitary pads. The pads are collected from schools, offices, and institutions using safe disposal bins. The waste is then treated using a hygienic and environmentally responsible process instead of being dumped.

Process:
PadCares system ensures that used sanitary pads are safely handled, treated, and converted into recycled material. This reduces landfill waste, prevents pollution, and improves working conditions for handlers.

Impact:
By offering a sustainable alternative, PadCare helps reduce environmental damage while improving menstrual hygiene awareness. Their work also supports safer working conditions for people handling sanitary waste.

Who they work with:
PadCare collaborates with schools, offices, factories, and institutions to promote responsible menstrual waste disposal and better hygiene practices.

from sentence_transformers import SentenceTransformer
import faiss
import pickle

# Load embedding model
embedder = SentenceTransformer("all-MiniLM-L6-v2")

# Read cleaned text
text = open("padcare_gpt/data/raw.txt").read()

# Split text into smaller chunks for better search
chunks = []
chunk_size = 300  # characters per chunk
for i in range(0, len(text), chunk_size):
    chunks.append(text[i:i+chunk_size])

# Create embeddings
embeddings = embedder.encode(chunks)

# Build FAISS index
index = faiss.IndexFlatL2(embeddings.shape[1])
index.add(embeddings)

# Save index and chunks
faiss.write_index(index, "padcare_gpt/index/index.faiss")
pickle.dump(chunks, open("padcare_gpt/index/chunks.pkl", "wb"))

print(" index ready ")

def padcare_chat(question):
    """
    PadCare GPT: Answer questions about PadCare Labs in a friendly, natural, and human way.
    - Simple questions: 1-3 paragraphs
    - Summaries: bullet points
    - Posts/pitches: lively and professional
    - Always stay true to the provided information, do not hallucinate.
    """

    # Convert question to embedding vector
    q_vec = embedder.encode([question])

    # Search top 2 relevant chunks
    _, ids = index.search(q_vec, 2)
    context = " ".join([chunks[i] for i in ids[0]])

    # Build a natural, human-friendly prompt
    prompt = f"""
You are PadCare GPT  a smart, friendly assistant who explains things clearly and naturally.
Use ONLY the information below. Avoid repeating sentences. Make it simple, human, and easy to read.
- For simple questions: 1-3 paragraphs
- For summaries: bullet points
- For posts or pitches: lively, professional, friendly

--- INFORMATION START ---
{context}
--- INFORMATION END ---

Question: {question}
"""

    # Tokenize the prompt
    inputs = tokenizer(prompt, return_tensors="pt")

    # Generate the answer
    outputs = model.generate(
        **inputs,
        max_new_tokens=400,    # Enough length for paragraphs
        temperature=0.7,       # Makes output more natural
        do_sample=True,
        top_p=0.9,
        top_k=50,
        no_repeat_ngram_size=2
    )

    # Decode tokens into human-readable text
    answer = tokenizer.decode(outputs[0], skip_special_tokens=True)
    return answer
